Allows to sent nofitications via IM.

Options for notifications:

* Never
* Only IM (if online)
* IM (if online) + email (if offline)
* IM (if online) + email 
* Only Emails

Tested on Odoo 8.0 ab7b5d7732a7c222a0aea45bd173742acd47242d
